# Tool Action Catalog (Agent Callable)

Định nghĩa toàn bộ tool/action để bất kỳ agent nào (bao gồm agent sanic) có thể gọi qua `tool_name` + `args` một cách nhất quán.

**Nguyên tắc chọn Tool:**
1. **Luôn dùng Tool của LangChain** (ví dụ: `GmailSendMessage`, `SheetsUpdateValuesTool`) nếu có sẵn và đáp ứng yêu cầu.
2. Chỉ dùng **Custom API / Adapter** (gọi API Google trực tiếp) đối với những tính năng **LangChain chưa hỗ trợ**.

## Chuẩn action

```json
{
  "tool_name": "gmail.send_email",
  "version": "v1",
  "risk_level": "high",
  "requires_hitl": true,
  "args_schema": {},
  "returns_schema": {}
}
```

## 6) Risk & HITL policy mặc định

- Low (auto): `search`, `get`, `read`, `list`.
- Medium (policy-based): `create_draft`, `append_values`, `update_event`, `copy_file`, `move_file`.
- High (HITL required): `send_email`, `delete_file`, `share_file` (external), `docs.batch_update` (critical), `sheets.batch_update_values` (critical range).

## 7) JSON schema mẫu cho một số tool trọng yếu

### 7.1 `gmail.send_email`

Args:

```json
{
  "to": ["user@example.com"],
  "cc": [],
  "bcc": [],
  "subject": "string",
  "body": "string",
  "thread_id": "optional"
}
```

Returns:

```json
{
  "message_id": "msg_xxx",
  "thread_id": "thr_xxx",
  "status": "sent"
}
```

### 7.2 `calendar.create_event`

Args:

```json
{
  "calendar_id": "primary",
  "summary": "Weekly sync",
  "description": "optional",
  "start": "2026-04-02T09:00:00+07:00",
  "end": "2026-04-02T09:30:00+07:00",
  "attendees": ["team@company.com"],
  "timezone": "Asia/Ho_Chi_Minh"
}
```

Returns:

```json
{
  "event_id": "evt_xxx",
  "html_link": "https://calendar.google.com/...",
  "status": "confirmed"
}
```

### 7.3 `docs.batch_update`

Args:

```json
{
  "document_id": "doc_xxx",
  "requests": [
    {"insertText": {"location": {"index": 1}, "text": "Hello"}}
  ]
}
```

Returns:

```json
{
  "document_id": "doc_xxx",
  "revision_id": "rev_xxx",
  "applied": true
}
```

### 7.4 `sheets.batch_update_values`

Args:

```json
{
  "spreadsheet_id": "sh_xxx",
  "value_input_option": "USER_ENTERED",
  "data": [
    {
      "range": "Sheet1!A1:C2",
      "values": [["kpi", 123, "ok"]]
    }
  ]
}
```

Returns:

```json
{
  "spreadsheet_id": "sh_xxx",
  "total_updated_ranges": 1,
  "total_updated_cells": 3
}
```

### 7.5 `drive.search_files`

Args:

```json
{
  "query": "name contains 'report' and mimeType='application/vnd.google-apps.document'",
  "folder_id": "optional",
  "page_size": 20
}
```

Returns:

```json
{
  "files": [
    {"id": "file_xxx", "name": "Weekly Report", "mime_type": "application/vnd.google-apps.document"}
  ]
}
```
