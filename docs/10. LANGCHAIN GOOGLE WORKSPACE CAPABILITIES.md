# LangChain Google Workspace Capabilities (Context7 Verified)

## Mục tiêu

Tài liệu này tổng hợp những gì LangChain có thể làm với Gmail, Google Calendar, Google Docs, Google Sheets, Google Drive, dựa trên tài liệu chính thức đã kiểm tra qua Context7.

## Packages chính thức liên quan

- Python: `langchain-google-community`
- Python (Drive integration docs): `langchain-googledrive` + `langchain-community`
- JavaScript/TypeScript: `@langchain/community`

## Capability Matrix

### 1) Gmail

**Python (official tools):**

- `GmailCreateDraft`
- `GmailSendMessage`
- `GmailSearch`
- `GmailGetMessage`
- `GmailGetThread`

**JS/TS (official tools):**

- `GmailCreateDraft`
- `GmailSendMessage`
- `GmailSearch`
- `GmailGetMessage`
- `GmailGetThread`

**Ngoài đọc/viết cơ bản, có thể làm thêm:**

- Lấy toàn bộ thread hội thoại theo ngữ cảnh.
- Tìm kiếm nâng cao theo query Gmail.
- Tạo draft trước khi gửi để gắn HITL gate.

## 2) Google Calendar

**Python (official tools):**

- `CalendarCreateEvent`
- `CalendarSearchEvents`
- `CalendarUpdateEvent`
- `CalendarMoveEvent`
- `CalendarDeleteEvent`
- `GetCalendarsInfo`
- `GetCurrentDatetime`

**JS/TS (official tools):**

- `GoogleCalendarCreateTool`
- `GoogleCalendarViewTool`

**Ngoài đọc/viết cơ bản, có thể làm thêm:**

- Di chuyển event giữa các calendar.
- Lấy metadata nhiều calendar.
- Neo thời gian hiện tại theo timezone (`GetCurrentDatetime`) để lập lịch chính xác.

## 3) Google Sheets

**Python (official tools qua `SheetsToolkit`):**

- `SheetsCreateSpreadsheetTool`
- `SheetsGetSpreadsheetInfoTool`
- `SheetsReadDataTool`
- `SheetsBatchReadDataTool`
- `SheetsFilteredReadDataTool`
- `SheetsUpdateValuesTool`
- `SheetsAppendValuesTool`
- `SheetsClearValuesTool`
- `SheetsBatchUpdateValuesTool`

**JS/TS:**

- Chưa thấy trang integration Google Sheets chính thức trong docs JS mới (ghi nhận trạng thái 404 tại thời điểm kiểm tra).

**Ngoài đọc/viết cơ bản, có thể làm thêm:**

- Batch read/batch update thay vì thao tác từng range đơn lẻ.
- Filtered read để lấy dữ liệu theo điều kiện.
- Append và clear range theo workflow ETL/reporting.

## 4) Google Drive

**Python (official integration docs):**

- `GoogleDriveSearchTool`
- `GoogleDriveAPIWrapper`

**JS/TS:**

- Chưa thấy trang integration Google Drive chính thức trong docs JS mới (ghi nhận trạng thái 404 tại thời điểm kiểm tra).

**Ngoài đọc/viết cơ bản, có thể làm thêm:**

- Search file theo query template trong folder.
- Chuyển nhiều MIME (bao gồm Google Docs/Sheets/Slides) về `Document` để dùng cho agent/RAG pipeline.
- Giới hạn phạm vi truy vấn theo `folder_id` để tăng an toàn.

## 5) Google Docs

**Kết quả từ docs chính thức LangChain:**

- Chưa thấy toolkit/tool riêng chính thức cho Google Docs (Python/JS) trong nhóm tools hiện tại.
- Thực tế có thể xử lý nội dung Docs gián tiếp qua Drive integration (MIME `application/vnd.google-apps.document`) trong luồng search/load/convert.

**Ý nghĩa cho sản phẩm:**

- Nếu cần thao tác Docs nâng cao (cấu trúc block, format chi tiết, comment, suggestion mode), nên thiết kế adapter riêng gọi Google Docs API trực tiếp và bọc thành custom tools cho agent.

## Constraints và lưu ý triển khai

- Gmail tools có cảnh báo bảo mật vì có thể vừa đọc vừa thay đổi trạng thái (draft/send).
- Sheets: đọc public có thể dùng API key; ghi dữ liệu cần OAuth2.
- Drive: cần credential/token phù hợp, có thể cần thêm dependencies cho parse nhiều định dạng.
- Nên cố định chính sách HITL: mọi action `send`, `delete`, `share external`, `critical overwrite` phải duyệt bắt buộc.
